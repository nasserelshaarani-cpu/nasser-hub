# 🏀🚀 Nasser's Arcade — Portfolio Hub

A personal portfolio website that connects all three of my projects with an AI-powered NBA Coach chatbot!

**Made by Nasser @ BEMEA**

---

## 🎮 What's Inside

- **Basketball History** — Learn legendary basketball moments
- **Space Dodge** — Arcade space game with skins and leaderboards
- **NBA Quiz** — Test your NBA knowledge
- **NBA Coach AI** 🤖 — Chatbot that answers trivia, explains rules, and generates quiz questions

---

## ✨ Zero Setup Required!

The NBA Coach is powered by **Pollinations.ai** — a completely free AI service:

- ✅ **No API key needed**
- ✅ **No signup required**
- ✅ **No age verification**
- ✅ **Works instantly out of the box**

Just open `index.html` in your browser and the chatbot works! 🎉

---

## 🚀 How to Use

### Run Locally
Just **double-click `index.html`** — it opens in your browser. Done!

### Deploy to GitHub Pages
```bash
git add .
git commit -m "Portfolio hub with NBA Coach AI"
git push
```

Then go to **Settings → Pages → main branch → root folder → Save**
